# 42 Philosophers in Go

This project is an implementation of the 42 Philosophers task by using Go language.

## Build

```bash
go build -o philo
```

## Usage

./philo 
number_of_philosophers 
time_to_die 
time_to_eat 
time_to_sleep 
number_of_times_each_philosopher_must_eat(option)

## Examples

```bash
./philo 5 800 200 200

./philo 5 800 200 200 4
```
